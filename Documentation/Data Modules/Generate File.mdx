---
title: "Text to Document"
description: "Create a text document containing the specified content."
---

The `Text to Document` module is a data module that can convert text and images into a file.

You can use this module to transform your AI-generated text into a file of `PDF`, `DOCX`, `TXT`, `CSV` and `HTML` types

The `Text to Document` module has **variable inputs** and **one output**:

- **Input**: Inputs can be of type `text` and `image`; you can add them in any specific to structure the document as you see fit
- **Output**: `Document`, the output is of a file type in the specified format
